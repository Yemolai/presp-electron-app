Apagar base de dados

 Ao clicar neste botão toda a base de dados será destruída. Não há como desfazer esta ação ou recuperar dados apagados. Tenha certeza de realizar uma cópia de segurança através das opções de exportação indicadas acima antes de pressionar este botão. Uma confirmação será exibida ao clicar neste botão para evitar acidentes.
