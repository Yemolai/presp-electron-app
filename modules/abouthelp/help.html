Ajuda

PRESP é uma sigla que significa Programa para registro de entrada e saída de pessoas.

Este programa serve para registrar um controle rápido e prático de entrada e saída de pessoas para exportação em arquivo de texto (texto puro e CSV).

A página Registrar disponibiliza um formulário rápido de registro de pessoa para que o usuário adicione uma pessoa que está entrando ou saindo. A página Lista exibe todos os registros nos banco de dados. Enquanto a página Exportar permite que o usuário exporte os dados em texto puro e em CSV e que limpe o banco de dados.

Lembre-se de que apagar o banco de dados é uma ação permanente que não pode ser desfeita.

Para maiores detalhes, dúvidas ou reportar erros, problemas ou solicitar funcionalidades entre em contato com o desenvolvedor através dos dados exibidos no repositório Github do PRESP.
